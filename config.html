Instrucciones

Levantar el backend

  1. Instala Node.js 18+
  2. Desde la carpeta server/, ejecuta npm install y luego npm run dev
  3. El servidor corre en http://localhost:4000 por defecto

Configurar el frontend

  1. En esta pantalla, coloca el Backend Base URL (ej: http://localhost:4000)
  2. Elige la Fecha de inicio (por defecto 2025-08-01)
  3. Ajusta Días por vista y Años máximos si es necesario

Obtener credenciales (Service Account)

  1. Ve a "APIs y servicios" → "Credenciales"
  2. Crea "Cuenta de servicio"
  3. Descarga el archivo JSON de la clave
  4. Copia el email de la Service Account

Probar conexión

  1. Haz clic en "Probar Conexión"
  2. Si funciona, haz clic en "Guardar Configuración"

Configurar la aplicación

  1. Completa todos los campos del formulario
  2. Haz clic en "Probar Conexión"
  3. Si funciona, haz clic en "Guardar Configuración"
