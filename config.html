Instrucciones de Configuración

Crear proyecto en Google Cloud Console

  1. Ve a Google Cloud Console
  2. Crea un nuevo proyecto o selecciona uno existente
  3. Habilita la "Google Sheets API"

Obtener credenciales

  1. Ve a "APIs y servicios" → "Credenciales"
  2. Crea credenciales OAuth 2.0
  3. Configura orígenes autorizados: https://limos-app.github.io
  4. Copia el API Key y Client ID

Obtener ID del Google Sheet

  1. Abre tu Google Sheet
  2. Copia el ID de la URL: https://docs.google.com/spreadsheets/d/ID_AQUI/edit
  3. Comparte el sheet con tu email con permisos de "Editor"

Configurar la aplicación

  1. Completa todos los campos del formulario
  2. Haz clic en "Probar Conexión"
  3. Si funciona, haz clic en "Guardar Configuración"
